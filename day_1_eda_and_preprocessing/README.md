# Day 1: Exploratory Data Analysis & Data Preprocessing

## Exploratory Data Analysis (EDA)
Exploratory Data Analysis is an approach to analyzing data sets to summarize their main characteristics, often using visual methods. EDA helps in understanding the data, detecting outliers, and identifying patterns.

## Data Cleaning
Data cleaning is the process of correcting or removing erroneous records from a dataset. It ensures quality data for analysis and includes activities like duplicate removal, error correction, and standardization.

## Handling Missing Values
Missing values can significantly impact analysis. Techniques to handle missing values include:
- Removing records with missing values
- Imputation (mean, median, mode)
- Using algorithms that support missing values directly.

## Outlier Detection
Outliers are data points that differ significantly from other observations. Identifying outliers is important as they can skew results. Techniques include:
- Visual inspection (box plots, scatter plots)
- Statistical methods (Z-score, IQR)

## Feature Scaling
Feature scaling techniques help in standardizing the range of independent variables. Common methods include:
- Min-Max Scaling
- Standardization (Z-score normalization)

## Data Visualization
Visualization is key in EDA. Libraries like Matplotlib and Seaborn allow the creation of informative and attractive visualizations such as:
- Histograms
- Scatter plots
- Box plots

## Statistical Analysis
Statistical analysis involves collecting and scrutinizing data to discover patterns and trends. It provides insights into relationships and helps make decisions.
